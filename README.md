# GameAccAutoStopper | 游戏加速器自动暂停工具

一款基于 Python 开发的自动化工具，旨在监控游戏进程并在游戏关闭后，自动操作加速器（AK 加速器）暂停计费。支持开机自启、严格模式、双重视觉验证以及详细的运行日志。

## ? 主要功能

- **进程监控**：自动识别配置文件中指定的多个游戏进程。
- **自动暂停**：检测到游戏关闭后，自动唤醒加速器窗口并执行点击暂停操作。
- **双重视觉验证**：使用 OpenCV 图像识别（PyAutoGUI）点击按钮，并验证按钮状态是否成功从“暂停”切换为“恢复”（存在一定失误的概率，失误就用严格模式）。
- **严格模式 (Strict Mode)**：开启后，游戏关闭时仅弹窗提醒手动操作，不执行自动点击，确保计费安全。
- **开机自启**：程序启动时可自动写入 Windows 注册表实现后台常驻。
- **运行日志**：实时生成 `running_log.txt`，记录游戏启停及加速器操作状态。
- **高分屏兼容**：支持 Windows DPI 缩放，确保在 2K/1K 屏幕上定位准确。

## ? 快速开始

### 1. 准备工作
在程序同级目录下准备好按钮截图：
- `btn_action.png`：加速器处于“进行中”时的**暂停按钮**截图。
- `btn_verify2.png`：加速器处于“已暂停”时的**恢复按钮**截图。

### 2. 配置说明
创建 `config.ini` 文件，格式如下：

```ini
[Settings]
# 监控的游戏进程名，用逗号分隔
game_processes = gta5.exe, cs2.exe, Cyberpunk2077.exe
# 加速器窗口的标题关键字
acc_title = AK加速器
# 监控频率（秒）
check_interval = 5
# 是否开启严格模式 (true/false)
strict_mode = false
